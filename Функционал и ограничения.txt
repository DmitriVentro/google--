Функционал для бека:
1. Запросы к БД
2. Получение токена
3. Получение списка документов (их выгрузка из папки)
4. Скачать выбранный документ на сервер
5. Удалить выбранный документ с аккаунта юзера
6. Подключиться к аккаунту другого выбранного юзера
7. Добавить документ с сервера на аккаунт выбранного юзера.
8. Удалить документ с сервера.

Ограничения системы:
1. У пользователя уже должен быть Гугл аккаунт
2. Система рассматривает только форматы  .doc .docx
3. Перемещать документ можно только между пользователями, зарегистрированными в нашей системе
4. Не сортируем документы
5. Не можем перемещать больше одного документика
6. Кнопка переместить блокируется, если идет перемещение документа.
