Инструкция по работе с репозиторием

  • Клонирование репозитория к себе

    1. Открываем cmd

    2. Выполняем команду:
      git clone https://github.com/DmitriVentro/google--.git

    3. Выполняем команду cd google--

    4. Отлично! Вы внутри!

  • Внесение изменений в локальном репозитории

      Добавление тестовых наработок на ветке dev

    1. В приложении Github Desktop

      1. Добавляем репозиторий, нажимая на кнопку Add -> Add existing repository
      2. Находим репозиторий на компьютере
      3. Меняем ветку на dev в current branch
      4. Даем название коммиту и коммитим

    2. В cmd

      1. Открываем cmd
      2. С помощью команды cd находим репозиторий
      3. Используем команду git
